# 獨立頁面規則

- 本目錄放各自獨立頁面或各功能自己專屬的規則。
- 這裡的規則不是全專案共用規範，也不是可重用樣板。
- 當某一頁面需要自己獨立維護初始化、事件、UI 行為或操作限制時，新增對應規則檔到本目錄。
- `hahaha_form_main.md` 放主頁自身處理方式；目前對應實作在 `project/hahaha_main_dll/form/`。
- `hahaha_form_test.md` 目前屬預留規則，只有測試頁重新加入 solution 時才套用。
- `hahaha_form_setting.md` 目前屬預留規則，只有設定頁重新加入 solution 時才套用。
